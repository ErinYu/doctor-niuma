## 5. 飞书卡片渲染优化（✅ 已完成 2026-03-12）

**问题：** 表格在飞书卡片中显示为原始 Markdown 文本而非渲染后的表格

**解决方案：**
- [x] 将 `sendMessage` 从 `msg_type: 'text'` 改为 `msg_type: 'interactive'`
- [x] 使用 `tag: 'markdown'` 元素替代 `tag: 'div'` + `tag: 'lark_md'`
- [x] 添加 `sendCard` MCP 工具支持
- [x] 在 IPC 层添加 `send_card` 消息类型处理
- [x] 在 router 层添加 `routeCardOutbound` 函数

**飞书 Markdown 限制说明：**
- 表格支持最多 10 列
- 每张卡片最多 5 个表格
- 只支持 H1 (`#`) 和 H2 (`##`) 标题
- 支持：粗体、斜体、代码、链接、列表、分割线

## 6. PPT 视觉多样化规则（✅ 已完成 2026-03-12）

**问题：** PPT 看起来"死板"，只用 content 布局

**解决方案：**
- [x] 添加布局多样化规则到 SKILL.md
- [x] 强制每 3-5 页插入 section 分隔页
- [x] 要求每种布局至少使用 1 次
- [x] 添加内容拆分规则（超过 6 个要点拆页）
- [x] 提供 15 页 PPT 视觉节奏模板

**布局使用场景指南：**
| 布局类型 | 使用场景 |
|----------|----------|
| section | 章节开始、主题切换 |
| two_col | 对比（药物A vs B、手术 vs 保守） |
| table | 数据展示（指南推荐、分级标准） |
| comparison | Before/After、方案对比 |
| quote | 指南原文、专家观点引用 |
| center_focus | 核心结论、关键数据 |
