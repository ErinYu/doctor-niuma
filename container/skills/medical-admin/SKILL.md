# 行政报告与统计自动化 (Medical Admin Skill)

该技能作为科室秘书，自动化处理科室行政事务、月度报表和 KPI 统计。

## 核心功能

1.  **KPI 自动提取与分析**：
    - 输入科室运行数据（Excel/CSV/截图）。
    - 自动提取门诊量、住院人数、周转率、药耗比、DRG/DIP 权重及盈亏。
    - 生成趋势图表描述（结合可视化工具时，可生成图表）。
2.  **科室/业务月报生成**：
    - 根据数据，自动生成结构化的“科室业务运行报告”。
    - 识别异常指标（如：本月并发症率上升）并进行风险提示。
3.  **多功能 PPT 大纲生成**：
    - 接收主题（如：科室质量控制、教学查房总结）。
    - 自动生成符合医院审美标准的 PPT 逐页大纲和讲稿。

## 使用工作流

### 第一步：数据接入与解析
识别输入的数据格式，如果是表格，利用 Python/Bash 工具进行结构化提取。

### 第二步：生成行政分析报告
```markdown
### [科室名称] [年份-月份] 业务运行简报
**核心指标摘要**：
- **门诊量**：[人次] (环比 [上涨/下降] [百分比])
- **平均住院日**：[天] (同比 [缩短/延长] [天数])
- **DRG 绩效**：CMI 指标 [数值]，入组率 [百分比]。
- **风险提示**：[如：非计划二次手术 [X] 例，高于平均水平，请关注]。
**重点工作建议**：
1. 优化 [具体流程] 以降低平均住院日。
2. 组织关于 [特定DRG分组] 的编码培训。
```

### 第三步：行政事务管理
- **值班/交班表生成**：根据医生名单和排班规则（如：主治每周一次总值班），自动草拟排班建议。
- **会议记录助手**：接收会议录音或零碎笔记，自动整理成标准的“科室会议记录”。

## 安全准则
- 行政报表不应包含具体患者的 PHI 信息，仅显示汇总和统计数据。
- 输出仅作为草案，最终由科室负责人审定。
