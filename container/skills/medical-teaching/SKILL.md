# 教学与多点执业支持 (Medical Teaching & Multi-practice Support Skill)

本技能帮助医生在繁忙的临床工作中，轻松应对教学查房、科室小讲课，并安全地在多个医疗机构间同步关键病例信息。

## 核心功能

1.  **医学教学辅助 (Teaching Assistant)**：
    - 输入一个具体病例特征。
    - 自动生成“教学查房/小讲课 PPT 大纲”。
    - 自动生成 3-5 道针对该病例的医学考题（包含正确答案与解析，参考规培/执医水平）。
2.  **值班/交班总结 (Handover Support)**：
    - 接收一整天的口头或文字笔记碎片。
    - 自动整理成标准的“SBAR 交班报告”（Situation, Background, Assessment, Recommendation）。
3.  **多点执业病例摘要 (Multi-practice Summary)**：
    - 接收复杂的病历文件。
    - 自动生成高度脱敏（去标识化）的“核心临床摘要”，方便医生在多点执业机构间安全沟通病情，确保符合 HIPAA 或 PIPL 要求。

## 使用工作流

### 第一步：识别教学/管理场景
- **教学场景**：根据病例，自动匹配最新的指南条文，生成教学课件逻辑。
- **临床管理场景**：整理当班期间的重点观察患者列表。

### 第二步：输出教学资料示例
```markdown
### 教学查房课件大纲 (针对 [疾病名称])
**第1页**：病例汇报 (脱敏信息)
**第2页**：临床特征与诊断思路
**第3页**：最新指南推荐方案 (引用来源)
**第4页**：互动环节 (病例延伸讨论)

### 随堂考题 (规培水平)
**问题 1**：[具体医学问题]
- A. [选项]
- B. [选项]
- C. [选项]
- D. [选项]
**正确答案**：[X]
**解析**：[根据临床指南的详细解释]。
```

### 第三步：生成标准交班报告 (SBAR)
当医生输入“交班：3床刘大爷心衰好转，8床刚收的...”时：
```markdown
### 值班/交班报告 (SBAR)
**S (现状)**：[X]床 [诊断]，目前生命体征 [稳定/波动]。
**B (背景)**：因 [主诉] 入院，既往有 [疾病史]。
**A (评估)**：目前主要问题是 [临床重点]，需关注 [指标]。
**R (建议)**：建议今晚监测 [指标/体征]，若出现 [症状] 则 [处理方案]。
```

## 安全准则
- **多点执业特别提醒**：在生成跨机构病例摘要时，必须剔除所有患者真实身份信息。
- **教学责任**：所有教学内容的知识库必须实时对接 `medical-literature` 技能中的最新指南，确保不传达过时医学知识。
