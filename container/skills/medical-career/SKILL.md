# 职称晋升 & 职业发展追踪器 (Medical Career Tracker Skill)

本技能帮助医生从繁重的职称晋升材料收集工作中解放出来，通过自动记录和对比要求，确保职业发展的每个节点都清晰可控。

## 核心功能

1.  **晋升条件自动化比对**：
    - 输入当前的职称（如：住院医师）和目标职称（如：主治医师）。
    - 输入所在省份/医院的晋升政策（或从 RAG 中获取）。
    - 自动生成“缺口清单”（Gap Analysis），例如：还差 30 例手术、2 篇 SCI、5 个 CME 学分。
2.  **病例/手术量自动累积**：
    - 自动从医生的日常病历处理记录中提取病例类型、手术难度分级。
    - 生成符合职称申报格式的“疑难病例汇总表”和“手术量证明材料”。
3.  **晋升材料包一键生成**：
    - 按照官方模板（PDF/Word 格式建议），自动填充简历、临床工作量、论文目录、基层服务证明摘要。

## 使用工作流

### 第一步：设定职业目标
记录医生的职称晋升时间表。

```markdown
### 职称晋升路线图 ([目标职称])
- **当前进度**：
  - [ ] 临床工作量：[完成数]/[要求数]
  - [ ] 学术论文：[完成数]/[要求数]
  - [ ] CME 学分：[完成数]/[要求数]
  - [ ] 基层服务时间：[完成数]/[要求数]
- **距离下一次评审**：约 [剩余天数] 天
- **待办预警**：[具体缺失项，如：急需完成 2 篇核心期刊投稿]
```

### 第二步：自动更新记录
当医生输入“这个病例很有代表性，记入我的晋升材料”时，自动提取关键要素。

### 第三步：生成申报草稿
当接收到“生成我今年的职称晋升临床总结”时：
```markdown
### 职称晋升临床工作总结 (草案)
**年度总工作量**：门诊量 [X] 人次，出院病人数 [Y] 人，独立完成三、四级手术 [Z] 例。
**疑难病例突破**：成功救治 [疾病名称] 等重症患者，其中 [具体病例摘要]。
**技术创新/教研**：开展了 [新技术名称]、带教 [人数] 名规培生。
```

## 数据合规
- 晋升材料涉及医生个人工作细节，必须严格隔离在 `medical-kb` 的长期记忆中。
- 生成的总结必须标注“仅为草拟，请根据真实情况修正”。
