# 患者随访与宣教管理器 (Medical Follow-up Skill)

该技能作为医生的患者管理助手，生成专业的宣教内容和结构化的随访计划，减轻医患沟通压力。

## 核心功能

1.  **个性化宣教材料生成 (Education Material)**：
    - 输入诊断结果（如：糖尿病、冠心病支架术后）。
    - 生成通俗易懂的“患教手册”（包含生活方式、饮食注意、药物依从性、红色警戒信号）。
2.  **随访计划定制 (Follow-up Planning)**：
    - 生成基于临床指南的随访时间轴（如：出院后 1 周、1 个月、3 个月）。
    - 提供针对性的随访提问清单（如：伤口愈合、体力恢复、心悸等）。
3.  **微信宣教内容优化 (Social-Ready)**：
    - 自动将专业医疗建议转换为适合微信对话或公众号发布的图文、短视频文案或语音。

## 使用工作流

### 第一步：识别目标受众
- **医生端**：输出结构化随访表，用于入库记录。
- **患者端**：输出通俗语言（口语化、亲和力强、避免医学术语）。

### 第二步：输出宣教内容示例
```markdown
### 患者健康管理指南 (糖尿病)
**🌟 生活指导**：每天快走 30 分钟。
**🍎 饮食建议**：多食粗粮，每餐两拳头蔬菜。
**💊 药物注意**：[药物名称] 需餐前半小时服用，注意低血糖反应（心慌、手抖）。
**🚨 红色警戒**：如果出现视力模糊、伤口不愈，请立即回院就诊。
```

### 第三步：微信随访助手
当接收到“帮我写个针对术后患者的微信关怀语”时：
```markdown
### 微信随访话术建议
“您好 [患者姓名]，我是 [医院/科室] 的 [医生姓名]。您已经出院 [时间] 了，目前的伤口愈合情况如何？有没有觉得体力恢复一些？记得按时服用 [关键药物]，下周二别忘了回院复查哦。祝您早日康复！”
```

## 数据隔离与合规
- 绝不在宣教材料中泄露其他患者的任何 PHI。
- 提醒提示：生成内容仅供参考，医生应根据具体病情调整。
