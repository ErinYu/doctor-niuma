# 多模态报告智能解读 (Multimodal Imaging Interpretation Skill)

该技能帮助医生快速“读懂”复杂的医学报告，支持跨时间维度的对比分析和异常自动识别。

## 核心功能

1.  **影像学报告关键信息提取 (Extraction)**：
    - 输入影像报告文字（如：CT、MRI、PET-CT）。
    - 自动提取病灶大小（如：1.2cm x 0.8cm）、位置（如：肺左下叶）、性质描述（如：磨玻璃影、边缘分叶）。
2.  **化验/检验结果趋势分析 (Trend Analysis)**：
    - 接收多份历史检验单（如：肿瘤标志物 CEA、肾功能指标）。
    - 自动生成数值趋势分析描述（如：“CEA 呈持续上升趋势，需关注复发风险”）。
3.  **智能影像总结与提示 (Insight)**：
    - 对比历史报告，自动标注“新发病灶”、“病灶增大”、“病灶缩小”或“无明显变化”。
    - 结合临床指南，给出复查频率建议。

## 使用工作流

### 第一步：多模态输入解析
当医生发送报告截图或 PDF 时，利用内置 OCR 或视觉模型进行解析。

### 第二步：生成专业解读报告
```markdown
### 影像/检查报告智能解读
**主要异常发现**：
- [位置]：发现 [性质描述]，大小约 [数值]。
- [性质]：[磨玻璃/实性/囊性] 结节。
**动态对比 (vs [前次日期])**：
- [ ] 病灶缩小 [百分比]
- [x] 病灶稳定，无明显变化
- [ ] 发现新病灶：[具体位置]
**辅助诊断建议**：
根据 [具体指南]，建议 [随访频率/进一步PET-CT检查]。
```

### 第三步：关键指标警示
当接收到“查看这次肝功能化验单”时：
```markdown
### 关键指标预警 ⚠️
- **ALT (谷丙转氨酶)**：[当前值] U/L (参考值：0-40)，**显著升高**。
- **AST (谷草转氨酶)**：[当前值] U/L (参考值：0-40)，**升高**。
**临床意义**：提示可能存在急性肝损伤，请结合患者用药史（如：抗结核药物）分析。
```

## 注意事项
- **必须声明**：解读结果仅作为辅助参考，最终诊断必须由放射科医生（阅片）和临床医生（结合病史）共同确认。
- **图像隐私**：处理截图时，自动忽略或提醒遮盖患者姓名、病案号等敏感区域。
